# WIND - Women in Democracy Website

This is a modern, responsive React website for the organization **WIND (Women in Democracy)**.

## Project Structure

This project was bootstrapped with [Vite](https://vitejs.dev/).

### Key Directories

- `src/components`: Reusable UI components like Header, Footer, Layout.
- `src/pages`: Individual page components (Home, About, Mission, etc.).
- `src/index.css`: Global styles and CSS variables.

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run the development server:**
    ```bash
    npm run dev
    ```
    Open your browser and navigate to `http://localhost:5173`.

3.  **Build for production:**
    ```bash
    npm run build
    ```
    The built files will be in the `dist` directory.

## Features

-   Responsive Navigation
-   Clean and professional NGO design
-   Component-based architecture
-   React Router for navigation
-   Lucide React icons
